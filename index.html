Butterfly - FFT
Group project for the exam "Sistemi Digitali Integrati": Hardware implementation of the butterfly part of a FFT algorithm. Following given specifications of timing, resources and optimisation. Tools used for project and test: ModelSim Altera 10.1d, Intel Quartus II 13.0, VHDL, Matlab, Excel.
Human Reflexes measurement device on FPGA
Group project for the exam "Sistemi Digitali Integrati": Hardware implementation on FPGA of a human reflex measurement tool. Tools used for project and test: ModelSim Altera 10.1d, Intel Quartus II 13.0, VHDL, Matlab.
Neural network node
Group project for the exam "Elettronica dei Sistemi Digitali": Hardware implementation of a neural network node. Tools used for project and test: ModelSim Altera 10.1d, Intel Quartus II 13.0, VHDL, Matlab, Excel.
